# TikTok Clone

TikTok風のショート動画アプリのUIクローン（学習目的）。
React Native + Expoを使ったモバイルアプリです。

> **注意**: このプロジェクトは学習目的で作成されたものであり、TikTokとは無関係です。

## 開発技術

- **Framework**: React Native + Expo
- **Language**: TypeScript
- **Routing**: Expo Router（タブナビゲーション）

## ローカルでの動かし方

```bash
npm install
npx expo start
```
